Plecos

Through an internship with the New York State Food Laboratory Division, I got to help their PDP department develop product codes. Originally, employees would have to manually generate product codes using an online tool. However, using some python and data science skills, I was able to automate a generator for these codes. By inputting a search phrase, the program I made would search the database for related items and automatically create the codes.
